# Setup Instructions for Medical Schedule Planner

This document outlines the installation and development instructions for the Medical Schedule Planner project.

## Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/{owner}/medical-schedule-planner.git
   cd medical-schedule-planner
   ```

2. **Install Dependencies**
   Make sure you have [Node.js](https://nodejs.org/) installed. Then run:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the root of the project and add the necessary environment variables. Refer to `.env.example` for required variables.

## Development

1. **Start the Development Server**
   To start the application in development mode, run:
   ```bash
   npm start
   ```

2. **Run Tests**
   To run the tests, use:
   ```bash
   npm test
   ```

3. **Build the Application**
   To create a production build, execute:
   ```bash
   npm run build
   ```
